## 项目综述

#### 项目工具：

> Tomcat + MySQL8 + JDBC8
>
> HTML + CSS + JS + JQuery
>
> 可能会添加的工具: AJAX XML等数据传输工具

#### 项目功能：

* 查询

1. 查询所有账单、分月查询账单、自定义区间查询账单
2. 账单分页
3. 关键词索引查询账单、账单金额查询账单
4. 综合多种查询要求查询账单
5. 自动根据动态数据生成图表

* 增删改

1. 添加账单
2. 删除指定账单
3. 修改指定账单
4. 多选删除

* 交互

1. 登录、注册

### 0.1版本功能

1. 查询所有账单、分类查询账单、按月查询账单
2. 添加账单
3. 删除指定账单（根据ID）
